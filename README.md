# Personal Documentation

Flat repository for guides, references, and learnings. Indexed by Pieces LTM for AI retrieval.

## Naming Convention

Format: `{category}-{topic}.md`

**Categories**:
- `guide-` - Tutorials, how-tos, best practices
- `ref-` - Reference materials, cheat sheets, specifications  
- `workflow-` - Process documentation, SOPs
- `config-` - Configuration documentation
- `notes-` - Meeting notes, dated entries (use ISO date: `notes-2024-10-19-topic.md`)

**Examples**:
- `guide-prompt-engineering.md`
- `ref-linear-api.md`
- `workflow-deployment.md`
- `config-zsh-setup.md`
- `notes-2024-10-19-planning.md`

## Usage

Store important context here + create Pieces memory linking to file for discoverability.
